Model Compression

Posted on February 13, 2023

To deploy a deep learning model in real-life, it is necessary to compress the model for real-time operation. There are several techniques for optimizing deep learning parameters, including pruning, knowledge distillation, and quantization.

  1. Pruning: This involves removing certain neurons or connections in a neural network that are found to be less important for its performance. The goal is to reduce the number of parameters in the model, which can make it faster and more computationally efficient. There are several types of pruning, including weight pruning and neuron pruning.
  2. Knowledge distillation: This is a technique for transferring knowledge from a larger, pre-trained model (the “teacher”) to a smaller model (the “student”). The student model is trained to mimic the outputs of the teacher model, with the idea being that the smaller model will have a similar level of accuracy while being more computationally efficient.
  3. Quantization: This involves converting the weights and activations of a neural network from high precision numbers (e.g., floating point numbers) to low precision numbers (e.g., 8-bit integers). This can reduce the memory requirements and computation time of the model, while still preserving its accuracy.

To properly implement pruning in real-life, you'll need to modify the model architecture to actually remove the weights and neurons that you've pruned. This can involve updating the forward pass of the model to exclude the pruned weights and neurons, and updating the optimizer to only update the remaining weights.

Here's an example of how you might implement weight pruning in PyTorch, with the added step of actually removing the pruned weights:
import torch
import torch.nn as nn
import torch.optim as optim

# Create a model
model = nn.Linear(10, 10)

# Train the model
# ...

# Choose a pruning criterion (e.g. the magnitude of the weights)
weights = model.weight.data.abs().clone().detach()
threshold = weights.mean() + 2 * weights.std()

# Get a mask indicating which weights to prune
mask = (weights >= threshold).float()

# Remove the pruned weights
model.weight.data *= mask

# Create a new optimizer that only updates the remaining weights
optimizer = optim.SGD(model.parameters(), lr=0.01)

# Zero-out the gradients of the pruned weights in the optimizer
for name, param in model.named_parameters():
    if 'weight' in name:
        param.grad *= mask
